    Merges the sequence of inputs with the existing history of inputs.

    Declaration
    public void MergeWith(uint timestep, ArraySegment<I> inputs)
    Parameters
    Type Name Description
    System.UInt32 timestep

    The start timestamp corresponding to the first element in the list.
    ArraySegment<I> inputs

    the list of inputs to merge into the history.
    Remarks

    Inputs from before the oldest recorded input will be ignored.

    Inputs newer than the newest recorded input will be appended to the history and become the newest inputs added.

    Intermediate inputs that have a matching input for the same timestep will be merged according to the provided during the history's construction.

    Intermedate inputs that do not have a corresponding input will be inserted into approriate locations.

    This operation runs in O(n) time where n is the number of elements managed by the history.
